Manage your WordPress site on the way using Telegram. Receive notifications, accept comments and edit WooCommerce orders.

== Description ==
After creating your personal Telegram bot, you can use the following features:
- Add the bot to multiple chats and groups
- Get notifications upon new posts and comments
- When WooCommerce is installed, you can also get notifications about new orders and reviews
- Edit and change status of posts, comments, orders, and reviews
- Browse through the latest posts, comments, orders, and reviews
In case you are missing any features or have found a bug, just let me know.
